Pro_Pack : tools : anim_binder
================================

ProPack anim_binder module is the replacement of the original AnimationBinder
module which was done back in 2011 for the Autodesk MasterClass support files.
This deals with creating the BND binder files themselves.

	>>> # import statement for the module via the r9pro decompiler
	>>> from Red9.pro_pack import r9pro 
	>>> r9anim_binder = r9pro.r9import('r9anim_binder')
